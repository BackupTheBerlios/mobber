Protocol

The initial commandset provides the following commands:

  • Login
    Log into the Server

    <mobber>
    <login>
    <name>THE USERNAME</name>
    <password>THE PASSWORD</password>
    </login>
    </mobber>

    Answer:
    <mobber>
    <login>
    <answer>OK</answer>
    </login>
    </mobber>

  • Logout
    Log out

    <mobber>
    <logout />
    </mobber>

    Answer:

    <mobber>
    <logout>
    <answer>OK BYE</answer>
    </logout>
    </mobber>

  • Listuser
    List all users currently online

    <mobber>
    <listuser />
    </mobber>

    Answer:

    <mobber>
    <listuser>
    <user>USER1</user>
    <user>USER2</user>
    <user>USERn</user>
    </listuser>
    </mobber>

  • Wall
    Message to all users

    <mobber>
    <wall>
    <mesg>THE MESSAGE</mesg>
    </wall>
    </mobber>

    Answer:

    <mobber>
    <wall>
    <sender>THE SENDER</sender>
    <mesg>THE MESSAGE</mesg>
    </wall>
    </mobber>

  • Message
    Message to a subset of users

    <mobber>
    <message>
    <recipient>USER1</recipient>
    <recipient>USER2</recipient>
    <recipient>USERn</recipient>
    <mesg>THE MESSAGE</mesg>
    </message>
    </mobber>

    Answer:

    <mobber>
    <message>
    <sender>THE SENDER</sender>
    <mesg>THE MESSAGE</mesg>
    </message>
    </mobber>

  • Listgroup
    List the available groups

    <mobber>
    <listgroup />
    </mobber>

    Answer:

    <mobber>
    <listgroup>
    <group>
    <name>GROUPNAME</name>
    <description>GROUP DESCRIPTION</description>
    </group>
    </listgroup>
    </mobber>

  • Joingroup
    Join a group

    <mobber>
    <joingroup>
    <group>GROUPNAME</group>
    </joingroup>
    </mobber>

    Answer:

    <mobber>
    <joingroup>
    <group>GROUPNAME</group>
    <answer>OK</answer>
    </joingroup>
    </mobber>

  • Partgroup
    Leave a group

    <mobber>
    <partgroup>
    <group>GROUPNAME</group>
    </partgroup>
    </mobber>

    Answer:

    <mobber>
    <partgroup>
    <group>GROUPNAME</group>
    <answer>OK</answer>
    </partgroup>
    </mobber>

  • Groupmessage
    Message to all users of the group

    <mobber>
    <groupmessage>
    <group>GROUPNAME</group>
    <mesg>MESSAGE</mesg>
    </groupmessage>
    </mobber>

    Answer:

    <mobber>
    <groupmessage>
    <group>GROUPNAME</group>
    <sender>SENDER</sender>
    <mesg>MESSAGE</mesg>
    </groupmessage>
    </mobber>
