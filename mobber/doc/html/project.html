Description

1.The Mobber

Mobber is an IRC-like Chat-Client that communicates with XML. It is written entirely in Java, so you can run it on every Plattform with a VM 1.2.

It is intended for chats on websites or as an easy to use conferencing/messaging tool. You can also build a "call-center" around Mobber.

2.Goals

Mobber should:
  • be easy to install
  • be easy to configure
  • have an expandable command-set
  • have plugable authentification modules (simple, mysql, LDAP ...)
  • have channels (read-only groups like stock-tickers)

3.License

4.Requirements

Mobber needs:
  • Apache projects ANT (Build tool)
  • Apache projects Log4J (Logger)
  • Apache projects Xerces1 (XML Parser)
  • JDOM (Abstraction layer for XML-Parsers)
  • Java 1.2
optional:
  • The mm.mysql-JDBC-Driver (for MySQL-Auhentification)
  • Sun JNDI and LDAP service provider (for LDAP-Auhentification)
